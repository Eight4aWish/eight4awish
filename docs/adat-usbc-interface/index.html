ADAT to USB-C Interface for Expert Sleepers ES-10

Project Overview

This project documents how to create an ADAT to USB-C interface to use the Expert Sleepers ES-10 8-channel Eurorack module as a computer audio interface. The ES-10 provides 8 channels of DC-coupled audio I/O via ADAT optical connection, making it perfect for interfacing modular synthesizers with your DAW.

What You'll Achieve:
  • 8 channels of high-quality audio I/O between your modular synth and computer
  • DC-coupled operation for CV and audio signals
  • Low latency performance suitable for real-time music production
  • USB-C connectivity for modern computers

About the Expert Sleepers ES-10

ES-10 Specifications

  • Channels: 8 input, 8 output (via ADAT optical)
  • Sample Rate: 44.1 kHz or 48 kHz (ADAT standard)
  • Coupling: DC-coupled (suitable for CV signals)
  • Format: ADAT Lightpipe optical (S/PDIF optical connectors)
  • Eurorack: 6HP width module
  • Power: +12V: 60mA, -12V: 50mA

Solution: USB Audio Interface with ADAT I/O

The most straightforward solution is to use a USB audio interface that has ADAT optical I/O. This eliminates the need for custom hardware and provides a reliable, professional-grade connection.

Hardware Connections

  1. Power: Connect the ES-10 to your Eurorack power supply (+12V, -12V, GND)
  2. ADAT Output: Connect ES-10's ADAT OUT (optical) to your USB interface's ADAT IN
  3. ADAT Input: Connect ES-10's ADAT IN (optical) to your USB interface's ADAT OUT
  4. USB: Connect your USB audio interface to your computer via USB-C
Cable Note: Use standard TOSLINK optical cables. These are the same cables used for S/PDIF digital audio. Cable length can typically be up to 5 meters without issues.

Software Configuration

Clock Synchronization

Critical: The ES-10 must be clocked from the USB audio interface. Set your interface to be the clock master and the ES-10 will automatically sync to the incoming ADAT signal.

Configuration Steps

On Your Audio Interface:

  1. Set the sample rate to 44.1 kHz or 48 kHz
  2. Set the clock source to "Internal" (the interface is the master)
  3. Enable ADAT input and output in the interface's control panel
  4. Set buffer size appropriately (128-256 samples is a good starting point)

In Your DAW:

  1. Select your USB audio interface as the audio device
  2. Enable the ADAT channels in your DAW's audio preferences
  3. Create audio tracks and assign them to the ADAT channels
  4. Set monitoring as needed (usually "auto" or "off" to avoid feedback)

DAW-Specific Notes:

  • Ableton Live: Preferences → Audio → Input/Output Config → Enable ADAT channels
  • Logic Pro: Preferences → Audio → Devices → I/O Buffer Size, Core Audio tab
  • Reaper: Options → Preferences → Audio → Device → Enable ADAT inputs/outputs
  • Bitwig Studio: Settings → Audio → Audio Device → Configure inputs/outputs
  • VCV Rack: Use the Audio module and select ADAT channels from your interface

Operating Voltage and Calibration

The ES-10 operates at Eurorack voltage levels:

  • Audio signals: ±10V peak (0dBFS in DAW = 10V peak in Eurorack)
  • CV signals: 0-10V or ±5V depending on application
  • DC-coupled: Can pass both audio and control voltages
Calibration: Expert Sleepers provides a calibration tool (Silent Way) that helps ensure accurate voltage scaling. This is especially important for pitch CV applications.

Alternative: DIY ADAT to USB Interface

For those interested in a DIY approach, you would need:

Components

  • USB Audio controller chip (e.g., XMOS XU216 or similar)
  • ADAT optical receiver (e.g., Cirrus Logic CS8416 or CS8427)
  • ADAT optical transmitter (e.g., Cirrus Logic CS8406 or Analog Devices ADM1032)
  • Optical TOSLINK connectors
  • ADC/DAC chips (if analog I/O is desired)
  • Supporting circuitry (power supply, crystal oscillators, etc.)
Complexity Warning: Building a custom USB audio interface is a complex undertaking requiring:
  • Advanced PCB design skills (4+ layer boards)
  • USB audio class driver implementation or ASIO driver development
  • Firmware programming for USB audio controllers
  • Clock domain management and jitter reduction
  • Extensive testing and debugging equipment

For most users, a commercial USB audio interface with ADAT I/O is far more practical and cost-effective.

Troubleshooting

No Signal from ES-10 to Computer

  • Verify optical cable is firmly connected and not damaged
  • Check that ADAT input is enabled in interface control panel
  • Confirm ES-10 is receiving power (check LED indicators)
  • Ensure interface is set as clock master (Internal clock source)
  • Try a different sample rate (44.1 kHz vs 48 kHz)

Crackling or Dropouts

  • Increase buffer size in DAW settings
  • Check CPU usage and close unnecessary applications
  • Verify USB cable is high quality and properly connected
  • Update audio interface drivers

Incorrect Voltage Levels

  • Use Expert Sleepers calibration tool (Silent Way)
  • Check gain settings in interface control panel
  • Verify DAW is set to unity gain (0dB) for CV applications
