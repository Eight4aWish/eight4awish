# Eight4aWish Eurorack DIY

Eurorack DIY designs including modules based around a range of microcontrollers, plus guides for integrating modular systems with computers.

## Projects

### Eurorack Modules
- **[84W-DaisyMFX](https://github.com/Eight4aWish/84W-DaisyMFX)** - Daisy Seed multi-effects module
- **[84W-ESP32ClkOsc](https://github.com/Eight4aWish/84W-ESP32ClkOsc)** - ESP32 oscillator module with clock mode

### Computer Interface Projects
- **[ADAT to USB-C Interface](84W-ADAT-USB-C/)** - Portable ADAT interface solution for Mac using Expert Sleepers ES-10. A lightweight alternative to Ableton Push 3 for connecting Eurorack to your computer via USB-C.

## Documentation
