## Project Summary

**What:** A documentation project for setting up an ADAT to USB-C interface to use with Expert Sleepers ES-10

**Why:** To provide a lightweight, portable alternative to Ableton Push 3 for connecting Eurorack modules to Mac computers

**Who:** Modular synthesizer enthusiasts, electronic musicians, sound designers using Expert Sleepers ES-10 or similar ADAT Eurorack modules

**Cost:** $250-$800 (depending on interface choice)

**Time to Setup:** ~30 minutes following the guide

**Result:** 8 channels of DC-coupled CV/audio I/O between your Mac and Eurorack system

## Key Features

✅ Comprehensive hardware recommendations  
✅ Step-by-step setup instructions  
✅ Visual connection diagrams  
✅ DAW configuration for Ableton Live and Logic Pro  
✅ Extensive troubleshooting guide  
✅ Budget options from $250-$800  
✅ Portable and lightweight (1-2 lbs vs 6.6 lbs for Push 3)  
✅ Low latency (~4-8ms typical)  
✅ Full 8-channel ADAT support
